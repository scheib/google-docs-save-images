Image Copy & Paste Saver

This page allows you to easily save images from sources like Google Docs, Sheets, and Slides that otherwise don't have download original image functionality.

Google Workspace apps (at time of authoring this page) don't have a way to save images, charts, or graphics. With this page you can copy images (even multiple at once), paste them here, and save them as files. This works from many other sources as well.

All processing happens in your browser: nothing is uploaded, and no data of any kind leaves your machine. This is a self-contained single HTML file.

Source code: github.com/scheib/google-docs-save-images. Hosted here: scheib.github.io/google-docs-save-images.

Copy an image (or multiple) from Google Docs and paste or drop here.
