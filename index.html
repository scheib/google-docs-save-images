Paste Google Docs Images

This page allows you to save images directly from Google Docs, Sheets, and Slides.

Google Workspace apps (at time of authoring this page) don't have a way to save images, charts, or graphics. With this page you can copy images (even multiple at once), paste them here, and save them as files.

All processing happens in your browser: nothing is uploaded, and no data of any kind leaves your machine. This is a self-contained single HTML file.

Source code: github.com/scheib/google-docs-save-images. Hosted here: scheib.github.io/google-docs-save-images.

Copy an image (or multiple) from Google Docs and paste here.
