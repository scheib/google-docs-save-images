# google-docs-save-images

A minimal [static web app for saving images copied from Google
Workspace](https://scheib.github.io/google-docs-save-images/) (Google Docs,
Slides, Sheets, Vids) editors.

## Why this exists

Google Workspace does not make it easy to save images directly from **Google
Docs**, **Google Slides**, or charts from **Google Sheets** as normal files on
your computer in their original quality. Taking a screen shot degrades image
quality. With this web app you copy those elements from an editor (one or
many), paste into the page, and save the results.

## Privacy and offline use

All operations run entirely in your browser. **No data is sent to any
server**—nothing is uploaded and nothing leaves your machine except the files
you choose to save. The app is a **single self-contained HTML file**; you can
keep a local copy and **use it offline**.

## Usage

Focus the page, paste from Google Docs (or Slides/Sheets where the clipboard
includes image data), then use the previews and download controls as needed.
Optional auto-download for PNG/JPEG is available and uses local storage to
remember the preference.
